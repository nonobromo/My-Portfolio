HTML+CSS Module Project.
פרויקט זה מסכם את כלל החומר הנלמד במודול הנוכחי של HTML/CSS. שילוב בכלים כמו flex,grid,position וכו...
הלוגו של העמוד לקח השראה מהשם של הקבוצה שבה הכיתה לומדת, וככה גם העיצוב שלו, בדגש על צבעים של שחור וירוק.
כל כפתור באיזור הניווט מוביל לחלקים השונים בעמוד, האזור המרכזי, הקצת עליי, הפרויקטים, ואזור צור הקשר. ישנו פוטר המוביל לכלל עמודי הרשתות החברתיות שלי כולל Linkedin.


ישנם חמישה תיקיות מרכזיות בהן הן מכילות את קבצי הפרויקט.
1.תיקיית SASS: תיקייה זו מכילה את כלל קבצי העיצוב לעמוד המרכזי של הפרויקט.
2.landingPages: תיקיית הפרויקטים המכילה שמונה פרויקטים, בכל אחת יש קבצי העיצוב והתמונות הרלווטניות.
3.previewPages: מכילה את כלל דפי התצוגה לפרויקטים, קבצי העיצוב והתמונות.
4.profilioImages: קובץ זה מכיל את כלל התמונות והאייקונים שיש בעמוד הראשי.
5. DownloadFiles: קובץ המכיל את כל קבצי ההורדה, הכולל את קורות החיים שלי בנוסף לפרויקטים בקובץ ZIP

*****הערות******
השימוש ב SASS עבור פרויקט זה נעשה אך רק לעמוד המרכזי, ללא שימוש במשתנים, אך ורק partials, לכל חלק חלק בעמוד.
השימוש ב Boostrap גם הוא רק בשביל העמוד הראשי, נעשה שימוש בקונטיינטר, אזור ניווט, וכפתורים אחידים שמתאימים מבחינה עיצובית לעמוד עצמו.
בעמודי הנחיתה, היה לי נוח יותר לפצל את קבצי העיצוב ל: קובץ עיצוב מרכזי, וקובץ עיצוב רספונסיבי, ולבסוף לקשר את שני הקבצים הללו לקובץ HTML. אותו הדבר גם לדפי התצוגה.
בעמודי נחיתה אין אזור ניווט, הלוגו באזור ההדר מוביל חזרה לעמוד הראשי.
